# 🚗 Car Price Prediction

A Machine Learning web application built using **Python**, **Streamlit**, and **Scikit-learn** to predict the selling price of a used car.

## Features
- Predicts car selling price
- User-friendly Streamlit interface
- Random Forest Regressor model
- Supports multiple car features

## Technologies Used
- Python
- Streamlit
- Pandas
- NumPy
- Scikit-learn

## Dataset
The project uses a car price dataset (`car_data.csv`) containing details like:
- Car Name
- Company
- Year
- Present Price
- Kilometers Driven
- Fuel Type
- Seller Type
- Transmission
- Owner
- Selling Price

## How to Run

1. Clone the repository
2. Install the required packages:

```bash
pip install -r requirements.txt
```

3. Run the Streamlit app:

```bash
streamlit run car.py
```
